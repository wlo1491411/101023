# 迷宮規則.2025/11/9

1. **遊戲說明**
   1. 遊戲完整度：此遊戲目前為測試階段，各方面都需要再打磨精細
   2. 遊戲核心：透過玩家提供角色、敵人、關卡等設定，由Gork依據規則自動進行遊戲與生成內容，最終呈現完整的冒險過程，並記錄歸檔，紀錄完成後可再使用成長過的角色進行新的關卡與冒險。
   3. 遊戲重點：角色成長、隨機性冒險、R18露骨的情色內容
2. **角色說明：角色會由玩家製作角色卡提供給Gork，並由角色卡中的角色進行遊戲。**
   1. 角色狀態：角色維持冒險的生理狀態。
      1. 意識值
         1. 基本值為5，為角色的意識，若歸零則角色會失去意識，該次冒險失敗。
         2. 每一場戰鬥階段或性愛階段結束，意識值會恢復1點。
         3. 每一次陷入高潮，意識值會減少1點。
      2. 高潮值
         1. 基本值為0，基本最高值為100，每次冒險中受到敵人的攻擊與性愛都會累積高潮值。
         2. 高潮值達到滿值時，角色會陷入高潮，並且意識值會減少1點。
         3. 當角色高潮時，會有一段陷入高潮的演出(約200字描述角色如何陷入高潮以及高潮時的反應。)
         4. 當角色在戰鬥階段中第一次陷入高潮時，將轉換為性愛階段。
   2. 角色技能：角色持有的技能，用於戰鬥與性愛。
      1. 戰鬥技能
         1. 近戰：近距離的兵器或搏擊的技能，如刀劍或格鬥。
         2. 遠戰：中遠距離射擊的技能，如弓弩或火器。
         3. 魔術：驅動魔力施展魔術的技能，如火球或冰矢。
         4. 身法：透過高速運動壓制的技能，如分身或閃現。
         5. 戰鬥技能數值與傷害：每個技能都會有技能值與傷害值，技能值用於檢定使用技能時是否成功，傷害值用於判斷技能使用成功後，對敵人造成多少傷害。
      2. 性愛技能
         1. 口腔：用嘴來替目標帶來歡愉的能力，如吸吮、舔舐、口交。
         2. 乳房：用胸部替目標帶來歡愉的能力，如乳交。
         3. 陰部：用小穴替目標帶來歡愉的能力，如磨蹭、容納陰莖。
         4. 後庭：用肛門替目標帶來歡愉的能力，如磨蹭、容納陰莖。
         5. 性愛技能數值與傷害：每個技能都會有技能值與傷害值，技能值用於檢定使用技能時是否成功，傷害值用於判斷技能使用成功後，對敵人造成多少傷害。
      3. 角色特技：角色專屬的技能，可透過通過關卡、擊敗菁英、擊敗魔王等壯舉來取得。
      4. 角色特徵：描述角色外觀、身材與性癖好。
3. **檢定說明：本遊戲的檢定均以骰子工具進行全隨機的判定**
   1. 技能檢定：用於戰鬥階段、性愛階段、結算成長階段時使用
      - 檢定方式：以角色的技能數值為標準並骰一次百面骰(1D100)，骰值高於數值則失敗，低於數值則成功。(範例：角色A的近戰技能值為50，進行近戰攻擊時將骰一次1D100，數值高於50則失敗換敵人回合，數值低於50則成功進行傷害判定)
   2. 傷害檢定：用於戰鬥階段、性愛階段使用
      - 檢定方式：當角色檢定成功時，代表技能使用成功，將進行傷害判定，依據技能值後括弧內的骰值與數值來檢定。(範例：角色A近戰技能檢定成功，在角色卡中角色A的近戰為50(1D2)，此時傷害檢定即為骰1D2，出來的數值即為對敵人的傷害。)  
4. **戰鬥階段與性愛階段說明：當角色進入關卡後就會遭遇敵人並進入戰鬥階段**
   - 戰鬥階段流程與規則
     1. 規則：回合制，固定我方角色先進行行動，再換敵人行動，敵我雙方所有人都行動過後，一個回合結束，持續到結束或轉換。
     2. 行動：
        1. 我方角色可參考敵方弱點來進行戰鬥技能選擇來攻擊(無法選擇性愛技能)，只能進行一次。
        2. 敵方角色通常會持有2-3種技能，隨機選擇一種技能使用，只能進行一次。
        3. 如果是一對多的戰鬥，一樣所有角色均進行一次行動後結束一回合。
     3. 結束或轉換
        1. 結束：敵方所有敵人HP歸零後戰鬥結束。
        2. 轉換：當我方角色高潮值集滿陷入高潮後，轉換為性愛階段。
   - 性愛階段流程與規則
     1. 規則：回合制，固定我方角色先進行行動，再換敵人行動，敵我雙方所有人都行動過後，一個回合結束，持續到結束。
     2. 行動：
        1. 我方角色可參考敵方性癖來進行性愛技能選擇來服侍(無法選擇戰鬥技能)，只能進行一次。
        2. 敵方角色通常會持有2-3種技能，隨機選擇一種技能使用，只能進行一次。
        3. 如果是一對多的性愛，一樣所有角色均進行一次行動後結束一回合。
     3. 結束：當角色意志值歸零或敵方HP歸零後結束戰鬥。
   - 勝利規則
     1. 關卡每一層戰鬥或性愛階段勝利後，角色高潮值重新計算，意志值恢復1點。
     2. 挑戰完關卡後，高潮值重新計算，意志值恢復滿值，並進入結算成長環節。
   - 冒險失敗
     - 根據敵人卡上的「被昏厥」進行1D6判定，高機率被送回城鎮，低機率迷失於迷宮中，該角色將暫時無法使用，直到該角色被救援。
   - 結算成長環節：完成關卡後，將會對角色進行獎勵升級，分為技能升級與傷害升級
     1. 技能升級：統計該角色在關卡中所使用過的所有技能，並對每一個使用過的技能進行一次技能檢定，檢定成功則技能值+1D3，失敗則沒事，如同使用後熟練而提升的概念。
     2. 傷害升級：統計該角色在關卡中成功造成過傷害的技能，並對每個成功造成傷害過得既能進行一次10%的檢定，成功的話可隨機選擇兩種升級。
        1. 最低傷害質提升1點(+總數)，例：1D2提升到1D2+1
        2. 最高傷害質提升1點(+面骰)，例：1D2提升到1D3
5. **敵人說明：關卡中登場阻擋角色前進，且意圖強姦主角的敵人**
   1. 敵人資訊：包含敵人的描述、弱點、性癖、技能、被昏厥後果等資訊
      - 敵人描述：敵人的外觀描述與危險程度，有利於內容情境生成。
      - 敵人弱點：用於戰鬥階段，戰鬥技能名稱後面的「+N」即為角色對該敵人造成傷害時的額外傷害加乘。
      - 敵人性癖：用於性愛階段，性愛技能名稱後面的「+N」即為角色對該敵人造成傷害時的額外傷害加乘。
      - 敵人技能：敵人技能通用於戰鬥與性愛階段，根據階段不同，抵禦使用的技能也不同
        1. 抵禦技能：當敵人使用技能時，技能描述中會有對應的戰鬥技能與性愛技能，角色抵禦敵人的該技能，則進行敵人技能描述中的指定技能檢定，成功則代表防禦或閃躲成功，失敗則要接受敵人技能所造成的高潮值累積；依照階段進行對應階段技能檢定，不可在性愛階段用戰鬥技能檢定，也不可在戰鬥階段用性愛技能檢定。
        2. 技能結果：敵人的技能通常會有兩種結果，第一種是單純的高潮值累積，第二種是施加特殊狀態。
      - 被昏厥：即角色與該敵人進行性愛階段中，意識值被歸零後的結果，會進行一次檢定，並根據結果來對照敵人卡上所寫的進行壞結局或遣返。
        1. 壞結局：角色會被困在迷宮中，暫時將無法使用該角色，比如被抓回去當生產工具或迷失在性愛中。
        2. 遣返：角色在性愛完後，被其他人救回城鎮。
